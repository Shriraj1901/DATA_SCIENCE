# 🧠 Jupyter Kernel Management

This repository demonstrates understanding of running, restarting, and interrupting Jupyter Notebook kernels.

---

## 🔹 Running Cells

Cells were executed one by one using **Shift + Enter**.

- Variables persist across cells  
- Execution order affects output  
- Kernel stores state until restarted  

---

## 🔹 Restarting the Kernel

Kernel was restarted using:

Kernel → Restart Kernel

### ✔ Observations

- All variables were cleared  
- Outputs were removed  
- Notebook required re-running from top  

This ensures reproducibility and clean execution.
